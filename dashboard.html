Example Usage

By default, the dashboard uses the hardcoded base path: https://mjain4321.github.io/report/. The file name can be provided via a query parameter, e.g., ?file=csvjson3.json.

For example, to load the file csvjson3.json, use:
https://yourdomain.com/dashboard9.html?file=csvjson3.json

If you wish to specify a custom file path (e.g., if your JSON file is hosted somewhere else), check the "Use custom file path" checkbox to enable the input box, and then enter the full URL (e.g., https://example.com/path/to/yourfile.json).

Note: If your file is stored locally (e.g., C:\Users\Mukul Jain\Downloads\csvjson3.json), you must first upload it to a publicly accessible server (such as GitHub Pages) and then use the resulting URL.
